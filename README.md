
Quatrefoil
----

> Render Three.js with Respo style code(experimental).

Demo http://repo.quamolit.org/quatrefoil/

### Patching operations

* [x] `add-element`
* [x] `remove-element`
* [x] `replace-element`
* [x] `add-children`
* [x] `remove-children`
* [x] `add-material`
* [x] `update-material`
* [ ] `replace-material`
* [ ] `remove-material`
* [x] `add-params`(partial)
* [x] `update-params`
* [ ] `remove-params`
* [ ] `add-events`
* [ ] `remove-events`

```bash
yarn
yarn watch

```
